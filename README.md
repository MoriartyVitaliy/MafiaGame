# МАФИЯ — онлайн игра

Веб-игра «Мафия» в стиле нуарного детективного досье. Играть можно с друзьями по сети — каждый открывает игру в браузере и заходит по коду комнаты.

## Как запустить локально

Нужен [Node.js](https://nodejs.org/) версии 18+ а также докер.

```bash
docker compose up -d

npm install
npm start
```

Откройте `http://localhost:3000` в браузере. Чтобы поиграть с друзьями в одной локальной сети (Wi-Fi), они могут зайти по вашему локальному IP, например `http://192.168.1.23:3000`.



- Минимум 4 игрока.
- Роли распределяются случайно: **Мафия**, **Мафия (Дон)**, **Доктор**, **Детектив**, **Мирные жители**, **Путана**, **Маньяк** (количество особых ролей растёт с числом игроков).
